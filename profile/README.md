We build privacy infrastructure for human connection - starting with a campus.

Hush Networks is a student-only social application that lets people express themselves anonymously within their university network. Every account is verified by a university email. Every confession is protected by design, not by policy. The trust comes from the protocol, not from us.

## Projects

### Hush - the application

A university-only anonymous confession app, live on Android and iOS. Students sign in with their official university email, browse verified campus profiles, and send confessions that are completely anonymous. Identity is revealed only if both people confess to each other - a double-blind mutual reveal. No one else finds out. No public profiles. No social pressure.

**Live** - [hushnetworks.in](https://hushnetworks.in) &nbsp;·&nbsp; [Google Play](https://play.google.com/store/apps/details?id=com.nsrlabs.hush) &nbsp;·&nbsp; [App Store](https://apps.apple.com/in/app/hush-networks/id6759811580)

### HushChain - the protocol

HushChain is an application-layer blockchain protocol built to be the identity trust layer beneath Hush - and eventually beneath any privacy-first social application.

The core idea: anonymity enforced at the infrastructure level, not by company policy. A centralised server can be hacked, compelled, or simply misused. A well-designed cryptographic protocol cannot.

**What the protocol implements**

- **Zero-knowledge identity anchoring** - social identities committed to chain as hashed cryptographic artifacts, unreadable by any party including the application
- **On-chain mutual-reveal consent** - a reveal executes only when both commit-reveal sequences are independently valid; the application reads the resolved mutual state, never plaintext identity
- **Decentralised social graph ownership** - no single server controls the connection graph; no single point of failure or coercion

**Current status**

Live-validated at a university hackathon hosted by Next Tech Lab. Protocol is now in refactor - temporary network surfaces taken offline while the core architecture is hardened for open-source release. Public RPC and faucet are intentionally down during this phase.

**What is coming**

Source code, architecture documentation, contributor guide, and a clear path for builders who want to extend or fork HushChain for their own privacy-first applications.
